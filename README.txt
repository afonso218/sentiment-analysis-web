# Information #
This is a webapp made in Python using Flask (Web) and TextBlob (Sentiment Analysis).

#REQUIREMENTS
 - Python

# TO RUN WEBAPP #
cd <webapp folder>
. venv/bin/activate
python app.py

# TO STOP WEBAPP #
<CTRL+C>
deactivate

# Demo #
Type a sentence and will be display a emoji and color or sentiment.
It also detect the language and translate to english.
